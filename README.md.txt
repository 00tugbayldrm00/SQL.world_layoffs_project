WORLD TECH LAYOFFS ANALYSIS (COVID-19 to present)

This Project analyzes layoffs in the techology sector between 2020 and 2023. We aim to clean the data first and then doing an Exploratory Data Analysis (EDA) to identify the trends and generate insights

Dataset used : https://www.kaggle.com/datasets/swaptr/layoffs-2022

Tools used : SQL

Project goals :

*Identifying companies where the whole of the recorded industry was laid off

*Identifying max/min total layoffs

*Identifying which industry had the most total layoffs

*Identifying which sectors lost the highest percentage of employees in each country

*Identifying the biggest layoffs across different years based on country and industry

*Identifying which companies had the highest total laif off numbers in different years

How to use : Open the SQL files to explore the project. All steps are documented in the files
